# 🔐 Password Generator

A dynamic and customizable password generator built using JavaScript.
This tool allows users to generate strong and secure passwords based on selected criteria.

---

## 🚀 Features

* 🔢 Adjustable password length (4–30 characters)
* 🔡 Include lowercase letters
* 🔠 Include uppercase letters
* 🔢 Include numbers
* 🔣 Include symbols
* ⚡ Real-time length display with slider
* 📋 One-click copy functionality
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

1. Select the character types (symbols, numbers, uppercase, lowercase)
2. Adjust the password length using the slider
3. Click **Generate Password**
4. Copy the generated password using the copy button

---

## 🧠 Logic Overview

* Character sets are defined for each type
* A combined pool is created based on user selection
* Random characters are selected using `Math.random()`
* Password is generated using a loop and displayed in the input field
